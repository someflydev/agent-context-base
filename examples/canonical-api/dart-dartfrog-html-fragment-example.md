# Dart Dart Frog HTML Fragment Example

This example shows the Dart Frog posture for HTMX-style HTML fragments:

- keep transport and fragment rendering easy to trace
- return stable fragment markers so partial updates stay testable
- keep the fragment contract small and deterministic
