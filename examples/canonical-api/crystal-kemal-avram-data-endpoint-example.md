# Crystal Kemal Avram Data Endpoint Example

This example shows the preferred Crystal shape for a visualization-ready dataset endpoint:

- keep the route surface obvious in Kemal
- fetch chart points through an Avram query boundary
- emit a stable JSON payload that charting clients can consume directly
