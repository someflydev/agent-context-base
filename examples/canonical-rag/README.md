# Canonical RAG Examples

Use this category for preferred local ingestion, indexing, retrieval, and answer flow patterns.

## A Strong Canonical RAG Example Should Show

- tiny deterministic corpus
- chunking and metadata shape
- explicit index or collection creation
- one ingest-plus-query smoke path
- minimal real-infra integration path for the vector or search backend

## Choosing This Example

Choose this category when the repo adds local retrieval or question-answer behavior over indexed content.

## Drift To Avoid

- many retrieval strategies in one example
- examples that depend on large opaque corpora
- no distinction between smoke validation and backend integration verification
