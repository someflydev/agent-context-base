# examples/derived

This directory contains higher-level prompt and coverage fixtures used by derived repo generation and verification.

## What It Covers

- `example-prompts.yaml`: 100 single-project prompts
- `derived-examples.yaml`: derived prompt-first orchestration scenarios
- `spin-outs.yaml`, `tier-rankings.yaml`, `orchestration-strategies.yaml`: higher-order analysis used by verification

## Current Generated Repo Shape

Derived repos now keep their local operating bundle under `.acb/`, including:

- `.acb/manifests/project-profile.yaml`
- `.acb/.generated-profile.yaml`
- `.acb/specs/*.md`
- `.acb/validation/CHECKLIST.md`
- `.acb/validation/COVERAGE.md`
- `.acb/INDEX.json`

Derived repos use `.prompts/` plus `.acb/` as the startup surface. `PROMPTS.md` is not generated by default.
