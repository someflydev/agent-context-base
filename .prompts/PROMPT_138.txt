BACKGROUND: JVM PATTERNS FOR JWT AUTH
================================================================================

Java and Kotlin share the JVM and JJWT (io.jsonwebtoken), so their token
issuance and validation code is closely related. The key differences are in
how auth context is wired into the request lifecycle:

Java (Spring Boot):
  - A custom OncePerRequestFilter extracts and validates the JWT.
  - Auth context is stored in Spring's SecurityContextHolder — the standard
    Spring Security mechanism even when not using the full Spring Security
    OAuth2 stack.
  - Route permissions are enforced via a custom HandlerInterceptor or via
    method-level @PreAuthorize annotations.
  - The canonical example uses a custom filter + method-level @PreAuthorize
    with Spring Security's @EnableMethodSecurity. This is the
    minimal-but-real Spring pattern. No Spring Security OAuth2 server.

Kotlin (http4k):
  - http4k uses a Filter chain for middleware — a composable, functional style.
  - A JwtFilter wraps routes that need authentication.
  - Auth context is passed via RequestContext extension functions or
    via a custom lens on the Request object.
  - Route permissions are checked by a RbacFilter factory that wraps
    individual route handlers.
  - http4k's routing is functional — routes are values, not annotations.

JJWT library notes:
  - JJWT (io.jsonwebtoken) fluent API:
    Jwts.builder().subject(sub).claim("permissions", perms)
        .expiration(Date(...)).signWith(key).compact()
  - Parsing:
    Jwts.parser().verifyWith(verifyKey).build().parseSignedClaims(token)
  - JJWT v0.12.x changed the API significantly from 0.11.x.
    Target v0.12.x in the canonical example. Document the API version clearly.
  - java-jwt (Auth0) is the noted alternative; it has a different API style
    but similar capabilities. Do not mix the two in one example.

  JwtFilter.java (OncePerRequestFilter):
    1. Extract Bearer token from Authorization header; continue chain if missing
       (let Spring Security return 401 for protected endpoints)
    2. Parse token with JwtService.parseToken(); on failure, set 401 response and return
    3. Load user from store by sub; if not found, 401
    4. Verify acl_ver match; if stale, 401
    5. Verify tenant membership; if inactive, 403
    6. Build AuthContext; set in SecurityContextHolder via:
       UsernamePasswordAuthenticationToken auth = new UsernamePasswordAuthenticationToken(
         authContext, null, List.of());
       SecurityContextHolder.getContext().setAuthentication(auth);

  auth/JwtMiddleware.kt:
    fun jwtMiddleware(store: InMemoryStore, verifyKey: SecretKey): Filter =
      Filter { next ->
        { request ->
          val token = request.header("Authorization")
            ?.removePrefix("Bearer ")?.trim()
            ?: return@Filter Response(Status.UNAUTHORIZED)

          val claims = try {
            Jwts.parser().verifyWith(verifyKey).build()
              .parseSignedClaims(token).payload
          } catch (e: JwtException) {
            return@Filter Response(Status.UNAUTHORIZED)
          }

          val user = store.getUserById(claims.subject)
            ?: return@Filter Response(Status.UNAUTHORIZED)

          if (claims.get("acl_ver", Int::class.java) != user.aclVer) {
            return@Filter Response(Status.UNAUTHORIZED)
          }

          val tenantId = claims.get("tenant_id", String::class.java)
          if (tenantId != null && !store.verifyMembership(user.id, tenantId)) {
            return@Filter Response(Status.FORBIDDEN)
          }

          val ctx = AuthContext(
            sub = claims.subject,
            email = user.email,
            tenantId = tenantId,
            tenantRole = claims.get("tenant_role", String::class.java),
            groups = (claims.get("groups", List::class.java) as? List<String>) ?: emptyList(),
            permissions = (claims.get("permissions", List::class.java) as? List<String>) ?: emptyList(),
            aclVer = claims.get("acl_ver", Int::class.java),
            issuedAt = claims.issuedAt.toInstant(),
            expiresAt = claims.expiration.toInstant()
          )
          next(request.withAuthContext(ctx))
        }
      }

  auth/RbacFilter.kt:
    fun requirePermission(permission: String): Filter =
      Filter { next ->
        { request ->
          val auth = try { request.authContext } catch (e: Exception) {
            return@Filter Response(Status.UNAUTHORIZED)
          }
          if (!auth.hasPermission(permission)) Response(Status.FORBIDDEN)
          else next(request)
        }
      }

    fun requireSuperAdmin(): Filter =
      Filter { next ->
        { request ->
          val auth = try { request.authContext } catch (e: Exception) {
            return@Filter Response(Status.UNAUTHORIZED)
          }
          if (auth.tenantRole != "super_admin") Response(Status.FORBIDDEN)
          else next(request)
        }
      }

  auth/TokenService.kt:
    fun issueToken(user: User, store: InMemoryStore, signingKey: Key): String {
      val permissions = store.getEffectivePermissions(user.id)
      val groups = store.getGroupsForUser(user.id, user.tenantId ?: "").map { it.slug }
      val now = Date()
      val exp = Date(now.time + 900_000L)
      return Jwts.builder()
        .subject(user.id)
        .issuer("tenantcore-auth")
        .audience().add("tenantcore-api").and()
        .claim("tenant_id", user.tenantId)
        .claim("tenant_role", store.getMembershipRole(user.id, user.tenantId))
        .claim("groups", groups)
        .claim("permissions", permissions)
        .claim("acl_ver", user.aclVer)
        .claim("jti", UUID.randomUUID().toString())
        .issuedAt(now)
        .notBefore(now)
        .expiration(exp)
        .signWith(signingKey)
        .compact()
    }

  Main.kt — route composition:
    val routingHandler = routes(
      "/health" bind GET to healthRoutes.handler(),
      "/auth/token" bind POST to authRoutes.tokenHandler(store, signingKey),
      // Protected routes wrapped with jwtMiddleware:
      routes(
        "/me" bind GET to meRoutes.handler(store),
        "/users" bind GET to (requirePermission("iam:user:read") then usersRoutes.listHandler(store)),
        // ... all routes per registry
      ).withFilter(jwtMiddleware(store, verifyKey)),
      // Admin routes wrapped with both jwtMiddleware and requireSuperAdmin:
      routes(
        "/admin/tenants" bind GET to adminRoutes.listTenantsHandler(store),
        "/admin/tenants" bind POST to adminRoutes.createTenantHandler(store),
      ).withFilter(jwtMiddleware(store, verifyKey)).withFilter(requireSuperAdmin())
    )
    ServerFilters.RequestTracing()
      .then(routingHandler)
      .asServer(Jetty(8080))
      .start()
