# Assistant Memory Anchors

These anchors are compact reminders for long-lived sessions.

Use them when you need a small, repeatable memory surface without reloading a full doctrine bundle.

## Available Anchors

- `context/anchors/session-start.md`
- `context/anchors/context-integrity.md`
- `context/anchors/compose-isolation.md`
- `context/anchors/prompt-first.md`

## Usage Rule

Load an anchor only when it sharpens the current task. Anchors are reminders, not substitutes for workflows, stacks, or manifests.
