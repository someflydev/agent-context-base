# Session Start Anchor

- start with `AGENT.md`, `CLAUDE.md`, and `manifests/project-profile.yaml`; add `README.md`, `docs/repo-purpose.md`, `docs/repo-layout.md`, and `docs/session-start.md` when those front docs exist and are current
- infer the task before loading stack detail
- prefer one workflow, one archetype, and one canonical example
- use `python scripts/prompt_first_repo_analyzer.py <repo>` when repo signals are unclear
- stop when ambiguity would force multiple primary bundles
